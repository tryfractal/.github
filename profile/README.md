**Fractal transforms AI from a reactive chat box into a continuous operational system** — one that remembers, adapts, and acts around the way you work.

Instead of starting from zero every session, Fractal accumulates context, learns patterns, organizes work, activates the right agents, and reshapes the workspace around your current objective.

## 🌱 What Makes It Organic

| Property | Description |
|----------|-------------|
| 🧠 **Persistent Intelligence** | Memory, decisions, and patterns survive across sessions |
| 🔀 **Adaptive Workspace** | Interface and workflows change based on your domain and intent |
| ⚡ **Agent Orchestration** | Coordinated teams of specialized agents under your supervision |
| 🔧 **Real Execution** | Agents operate on real files, tools, tasks — not trapped in a text box |

## 🚀 Quick Start

1. **Download** Fractal from [Releases](https://github.com/tryfractal/fractal/releases)
2. **Create** your first workspace
3. **Chat** with the Orchestrator — your persistent AI companion
4. **Install** skills from the [Registry](https://github.com/tryfractal/registry)
5. **Build** — create tasks, collections, and automate workflows

## 🧩 Skills

Skills are installable system capabilities — not just prompts. A skill can bring:

- **MCP Server connections** — connect to GitHub, Notion, Slack, databases...
- **Collections** — domain models and data structures
- **Views** — custom dashboards and interfaces
- **Agents** — specialized AI assistants
- **Templates** — pre-built task workflows

Browse the [Registry →](https://github.com/tryfractal/registry)
